# Okurmen Test Platform

Онлайн-тест для оценки логики и мышления студентов учебного центра «Окурмен».

## Функциональные возможности

### Для студентов:
- Регистрация и авторизация
- Выбор уровня теста (легкий/средний)
- Прохождение теста с таймером (20 минут)
- Просмотр результатов с цветовой индикацией
- История прохождения тестов

### Для администраторов:
- Управление вопросами (CRUD)
- Просмотр всех пользователей
- Просмотр результатов тестирования
- Двуязычная поддержка (RU/KG)

## Технологический стек

### Backend:
- Node.js + Express
- PostgreSQL
- JWT для аутентификации
- Bcrypt для хэширования паролей

### Frontend:
- React + TypeScript
- Material-UI для интерфейса
- React-i18next для локализации
- Axios для HTTP-запросов
